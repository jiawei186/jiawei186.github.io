Python-based-2

1 赋值语句

a, b = b, a+b中, 先运算等号右边的a+b, 得到一个c
再将b赋给a, 即a=b
再将c赋给b, 即b=c

2 判断语句

对应的c为True时, 就执行对应的s, 从上到下判断
所有的c都为False时, 就执行else对应的s

else部分可省略

2.1 嵌套判断语句

一个判断语句里, 可以嵌套另一个判断语句

3 循环语句

3.1 while循环语句

s0是初始化语句, 只需执行一次

c1是判断条件语句
当c1为True时, 执行s1和s2, 并再次执行c1
当c1为False时, 执行s3, 并往下执行

s3是更新条件语句, 和s2区分开来

else部分可省略

无限循环
当判断条件语句永远为真时, 该循环为无限循环

简单while语句
省略else部分
s2和s3加起来只有一条语句, 将该语句放在c1后

3.2 for循环语句

st0是对se1的初始化

v1会逐一取se1中的元素
当v1成功取到se1中的元素时, 执行st1, 并再次执行v1
当v1取se1中的元素失败时, 执行st2, 并往下执行

else部分可省略

3.3 break语句

break是一个关键字, 单独成句

在循环语句里加入一个判断语句
在该判断语句里加入break语句
当c1为True时, 执行break, 该循环体会被跳出, 不再执行v1

3.4 continue语句

continue是一个关键字, 单独成句

在循环语句里加入一个判断语句
在该判断语句里加入continue语句
当c1为True时, 执行continue, 该循环体剩下的部分st2被跳过, 再次执行v1

3.5 pass语句

pass是一个关键字, 单独成句

pass语句不做任何事情
和空行起到的作用类似, 比空行多了一个占位的功能

3.6 range语句

range()可作为一个常用的sequence被用于循环语句中

range()函数中内置一个数字序列, 从0开始, 递增量为1
0的索引是0, range(5)=[0, 1, 2, 3, 4]

如果使用range(s, e, i)格式
那就可以不使用range()内置的数字序列, 可以自己制定一个新序列
例如range(-10, -100, -30)

可以使用range()和len()使得for循环语句内的v不再直接指代元素, 而是指代元素的索引
len()获得一个序列的元素个数

4 迭代

迭代器是一个可以记住遍历的位置的对象

迭代器对象从集合的第一个元素开始访问, 直到所有的元素被访问完成

迭代器只会往前不会后退

迭代器分为iter()和next()两部分

在for循环语句中使用迭代器, 就不需要设置条件更新语句了

4.1 类迭代

__iter__()方法返回一个特殊的迭代器对象
这个迭代器对象实现了__next__()方法并通过StopIteration异常标识来标识迭代的完成

StopIteration
StopIteration异常被用于表示迭代的完成, 防止无限循环的出现
raise StopIteration是在代码中被使用的语句

__next__()方法会返回下一个迭代器对象

4.2 生成器

yield是一个关键字, 其后跟着一个变量yield a或者表达式yield a+b而成句
包含yield语句的函数, 即为生成器函数

yield return
返回后面的值 返回后面的值
返回之后, 函数内的变量不会被消除 返回之后, 函数内的变量会被消除
下次调用时, 从该语句的下一句开始 下次调用时, 从第一句开始
使用迭代器调用所在的函数 使用函数名调用所在的函数

5 函数

函数
函数是组织好的, 可重复使用的, 用来实现单一或相关联功能的代码段

内建函数
Python内部已有的函数, 不需要程序员来写

5.1 用户自定义函数

def是关键字, 后接函数标识符和圆括号和冒号
圆括号里有参数

return是关键字, 后接表达式作为函数返回值
return后面也可以不接表达式

5.2 函数调用

想要调用一个函数, 只需要直接写出函数名和圆括号和参数

5.3 参数传递

对象有类型, 变量没有类型
对象是被存储在存储器中的实际数据, 变量只是一个指针或对象的引用

不可变类型, 即值传递 可变类型, 即引用传递
整数, 字符串, 元组 列表, 字典

如果不可变类型在函数中被修改了值, 则Python会将变量指向另一个对象, 相当于原先的对象消失了

如果可变类型在函数中被修改了值, 则就是原对象被改变了, 变量指向的是同一个对象

位置参数
传入参数的顺序和声明参数的顺序相同
传入参数的数量和声明参数的数量相同
传入参数时可以带关键字

指定位置参数
传入参数的顺序和声明参数的顺序相同
传入参数的数量和声明参数的数量相同
传入参数时不能带关键字

声明参数中, 在两个逗号之间单单出现一个/, 则其前的所有声明参数都是指定位置参数

关键字
关键字就是声明参数时参数的名字

关键字参数
传入参数的前面必须加上关键字=
或者传入参数的标识符和声明参数的标识符相同

声明参数中, 在两个逗号之间单单出现一个*, 则其后的所有声明参数都是关键字参数

默认参数
声明参数里就带有默认值=默认值

元组不定长参数
不定长参数一般被放在声明参数的最后方
元组不定长参数前有一个星号*variable, 该声明参数是一个元组
该声明参数对应的传入参数和它后面的传入参数, 都被划入该元组中
该声明参数也可以没有对应的传入参数

字典不定长参数
字典不定长参数和元组不定长参数类似
只是该声明参数对应的传入参数和它后面的传入参数, 需按照a=1, b=2的格式输入, 并都被划入该字典中

5.4 匿名函数

lambda是一个关键字, 它是一个微型的函数, 相当于一个表达式
function-1 = lambda [argument-1 [, arguement-2, ...]]: expression-1

5.5 return

return是一个关键字, 被用于退出函数
return后面可以添加表达式作为函数返回值, 也可以不添加表达式直接返回

6 with语句

6.1 上下文管理器

上下文管理器
支持上下文管理协议的类对象

上下文管理协议
类对象包含方法__enter()和__exit__()

自定义上下文管理器
可以在类定义中人为地添加上下文管理协议
再在函数中使用该类, 再用with使用该函数
也可以直接用with使用该类

6.2 with表达式的工作原理

with expression-1 as variable-1:
执行e1, 生成上下文管理器
获取上下文管理器的__exit()方法, 并保存起来
调用上下文管理器的__enter()方法; 如果使用了as子句, 则将__enter__方法的返回值赋值给as子句中的v1
执行冒号后的代码块
如果执行过程中, 没有出现异常, 则以None作为参数调用__exit__(None, None, None)
如果执行过程中, 出现了异常, 则使用sys.exc_info得到的异常信息为参数调用__exit__(exc_type, exc_value, exc_traceback)
